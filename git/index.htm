  Git installation on Windows

  1. Download Git from here: https://gitforwindows.org/

  2. Run installer exe and complete installation. One setting to ensure is checked is the Git Bash Here option, this allows you to right click on a folder in Explorer and Git Bash straight into it.


  3. Next important feature is the default editor for Git, personally it's Visual Studio Code.


  4. Leave the defaults for the remaining options.

  5. Finally once installed, open Git Bash and run the following command to ensure Git is installed:
    git --version

  Github Commands



  1. On Github retrieve the URL of the Github repository that will be pushed to

  2. Back in Git bash, in the repository that has to be pushed, add the URL for the remote repository using this command:
    git remote add origin remote repository URL

  3. Confirm remote repository has been set correctly?
    git remote -v

  4. If all recent changes have been added and committed, then push to remote repository:
    git push origin master

  5. Finally check the repo on Github that all files have been pushed?
