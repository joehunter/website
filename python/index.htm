About

Detailed on these pages are a solution to convert a fixed width ASCII file to CSV. Once converted the file is imported into SQL Server. Python was chosen as it has a number of libraries for quickly parsing fixed width format.

 Requirements

The requirement was to convert a fixed width ASCII file into CSV format so it could be imported directly into SQL Server. Previously we used T-SQL to import the file directly using OPENROWSET. But the driver being used (Microsoft.ACE.OLEDB.12.0) has a 255 field limit which was exceeded with this file. Thankfully there is a SCHEMA INI file to accompany the fixed width file and this can be used to reference the columns in the fixed width file.
